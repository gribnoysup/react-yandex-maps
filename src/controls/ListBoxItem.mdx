Renders Yandex.Maps API [ListBoxItem][control] control

## Basic Usage

<Playground>
  <YMaps>
    <Map
      defaultState={{
        center: [55.751574, 37.573856],

        controls: [],
      }}
    >

        <ListBoxItem data={{ content: 'Moscow' }} />

    </Map>
